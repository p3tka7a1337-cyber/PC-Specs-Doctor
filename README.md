# 💻 PC Specs Doctor 

PC Specs Doctor е desktop приложение, което автоматично засича хардуера на компютъра ти и проверява дали може да подкара избрана игра.  
Освен това предлага **GPU upgrade препоръки, FPS оценка и health score**.

---

## 🚀 Features

- 🖥️ Автоматично разпознаване на CPU / GPU / RAM  
- 🎮 20+ популярни игри  
- 🧠 Проверка дали играта ще върви  
- 🎯 GPU recommendation система  
- 📊 Health score (performance rating)  
- 🔥 FPS estimation (Low / Medium / High / Ultra)  
- 🪟 Modern GUI (CustomTkinter dark mode)  
- ⚡ Progress bar за анализ
